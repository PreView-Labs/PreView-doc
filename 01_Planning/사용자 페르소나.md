# PreView AI 사용자 페르소나 문서

본 문서는 **PreView AI**의 핵심 타겟 사용자를 구체적으로 정의하여, 서비스 방향성과 UX/UI 설계, 기능 우선순위 설정의 근거를 제공합니다.
AI 기반 이력서 맞춤형 면접 코칭 플랫폼으로서, 우리의 사용자가 누구이며 그들의 합격에 대한 열망과 불안을 어떻게 해결할 수 있을지를 명확히 합니다.

* **작성자:** [고동현](https://github.com/rhehdgus8831)
* **작성일:** 2025-12-02
* **문서 버전:** v1.0.0

---

## 1. 대상 독자

* **기획자 / PM** – 사용자 니즈(취업 고민)를 기반으로 기능 기획 및 서비스 로드맵 수립
* **디자이너 (UX/UI)** – 이력서 업로드부터 피드백 확인까지의 매끄러운 사용자 경험 설계
* **프론트엔드 개발자** – 대화형 인터페이스 및 실시간 피드백 리포트 UI 구현
* **백엔드 개발자 / AI 엔지니어** – 사용자별 맞춤형 질문 생성 로직 및 프롬프트 페르소나 설계
* **신규 합류자** – PreView AI가 해결하고자 하는 취업 시장의 페인 포인트(Pain Point) 이해

## 2. 핵심 사용자군 요약

| 구분 | 페르소나명 | 주요 특징 | 주요 니즈 |
| :--- | :--- | :--- | :--- |
| A형 | **막막한 신입 취준생 (대학생)** | 면접 경험이 거의 없고, 자신의 경험을 어떻게 말해야 할지 모르는 사용자 | 내 이력서에서 어떤 질문이 나올지 미리 알고 싶음 |
| B형 | **전략적 중고 신입/이직러 (주니어)** | 실무 경험은 있지만, 기술적 깊이를 검증하는 압박 면접이 두려운 사용자 | 뻔한 질문이 아닌, 내 프로젝트를 파고드는 날카로운 질문 필요 |
| C형 | **효율 중심 부트캠프 수료생** | 단기간에 많은 면접을 준비해야 하며, 스터디 구할 시간이 부족한 사용자 | 언제 어디서나 즉시 연습하고 객관적인 피드백을 받고 싶음 |

## 4. 사용자 여정 맵 (User Journey Map)

| 단계 | 주요 행동 | 감정 / 고민 | PreView AI가 제공하는 해결 |
| :--- | :--- | :--- | :--- |
| ① 필요 인식 | 서류 합격 통보 또는 면접 준비의 막막함 | "내 프로젝트에서 뭘 물어볼까?" | 회원가입 없이 서비스 소개 확인, 쉬운 접근성 |
| ② 정보 입력 | 이력서(PDF) 업로드 및 직무 선택 | "개인정보 유출은 안 되겠지?" | 이력서 자동 파싱 및 핵심 경험 추출 시각화 |
| ③ 모의 면접 | AI 면접관과 질의응답 (채팅/음성) | "대답 못하면 어떡하지? (긴장)" | 실제 면접과 유사한 꼬리 질문 알고리즘, 실시간 타이핑 효과(SSE) |
| ④ 피드백 | 면접 종료 후 리포트 확인 | "내 점수는? 뭘 고쳐야 하지?" | 논리성/키워드/태도 점수화, 답변 수정 제안(Best Practice) |
| ⑤ 재도전 | 피드백 반영 후 재응시 | "아까보다 나아졌을까?" | 과거 기록과 비교하여 성장 추이 그래프 제공 |

## 5. 사용자 요구사항 (Needs & Expectations)

* **초개인화**: "모두에게 묻는 질문"이 아니라 "내 이력서에 적힌 프로젝트"에 대해 물어봐야 함.
* **실전 같은 긴장감**: AI가 너무 기계적이거나 친절하기만 하면 안 됨. 날카로운 반론(꼬리 질문)이 필수.
* **객관적 평가**: "좋았어요"가 아니라 "두괄식 구성이 부족합니다", "Redis 사용 이유가 빈약합니다" 같은 구체적 지적.
* **접근성**: 별도 프로그램 설치 없이 웹 브라우저에서 바로 실행되어야 함.
* **비용 효율성**: 대학생/취준생이 부담 없이 이용할 수 있는 가격 정책 (또는 무료).

## 7. 정리

> **PreView AI의 사용자는 단순히 기술을 배우는 사람이 아니라, '자신의 가치를 증명하고 싶은 사람들'입니다.**
> 이들은 AI 면접관과의 치열한 공방을 통해 자신의 경험을 논리적으로 무장하고, 실전 면접에서 누구보다 빛나는 지원자가 될 것입니다.
> PreView AI는 그들의 불안을 확신으로 바꾸는 페이스메이커가 됩니다.
